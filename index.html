Info

This website allows a user to use either the Plantuml or Mermaid.js text based UML definition.

Thanks to the Plantuml and Mermaid.js folks for making their code available!

SequenceDiagram.org and Chartmage are alternatives that share some aspects of the format. However, neither support ASCII output. SequenceDiagram is not open source, while Chartmage is does not work on Mobile.

Code for this website is located at github.com/khilnani/uml.ide.
