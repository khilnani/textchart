Info

This website allows a user to use either the PlantUML or Mermaid.js text based UML definition.

Thanks to the PlantUML and Mermaid.js folks for making their code available!

SequenceDiagram.org and Chartmage are great alternatives. Neither SequenceDiagram nor Chartmage support ASCII output. Additionally, SequenceDiagram is not open source and neither are mobile friendly - Chartmage does not support mobile phones. However, both do have unique features. Please take a look at each option to see which meets your needs best. TextChart is designed to be mobile first.

Unfortunately, there is no standardized text UML format. But, PlantUML and Mermaid have the strongest community support.

The source code for this website is located at http://github.com/khilnani/textchart. Feel free to suggest improvements or report bugs.
