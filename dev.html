Info

This website uses the PlantUML format of text based UML definition. Thanks to the PlantUML folks for making the processing server and code available!

To learn more please take a look at http://plantuml.com.

SequenceDiagram.org and Chartmage are alternatives that share some aspects of the format. However, neither support ASCII output. SequenceDiagram is not open source, while Chartmage is based on Mermaid.js.

Both Mermaid.js and PlantUML are good. Take a look at each to see which fits your needs.

Code for this website is located at github.com/khilnani/uml.ide.
